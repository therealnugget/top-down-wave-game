An extension of the physics system i made in c++.
The current commit is the latest working version so feel free to check it out.<br /><br />
**TO PLAY THE GAME**:<br />run "top-down-wave-game\x64\Debug\simple physics system.exe".
The physics system was built using a quadtree for the broadphase and SAT for the narrow phase with linear impulse collision responses. SDL2 was used for rendering. 
Press "WASD" to move, "O" to rotate and "CLICK" to attack<br /><br />
**IF YOU WOULD LIKE TO BUILD THE SOLUTION YOURSELF**:<br />Open the solution file at "top-down-wave-game/top-down-wave-game.sln" in visual studio and hit "f5"
Ctrl + Q to quit
